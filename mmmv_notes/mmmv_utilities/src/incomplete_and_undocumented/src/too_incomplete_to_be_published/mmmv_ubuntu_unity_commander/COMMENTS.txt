===========================================================================
==================  mmmv_ubuntu_unity_commander  ==========================
===========================================================================

The name of the mmmv_ubuntu_unity_commander is inspired by the 
classical Norton Commander and its open source software clone, the 
Midnight Commander.

---------------------------------------------------------------------------
                             Dependencies
---------------------------------------------------------------------------

On Ubuntu the dependencies can be installed by executing:

    apt-get install wmctrl ruby bash
